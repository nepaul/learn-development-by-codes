## Build Setup

```bash
# basic-like
cd basic-like

# install dependencies
npm install

open index.html in your browser
###########



# excellent-like
cd excellent-like
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build
```
