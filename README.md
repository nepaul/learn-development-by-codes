# Learn web development by demos

> "Talk is cheap. Show me the code."  -- Linus Torvalds 


## Getting started
```
git clone https://github.com/nepaul/web-demos.git
cd one-demo-dir
read the README.md in one-demo-dir
enjoy it..
```
