## Simple-Koa-App

### Description

A simple Koa App with koa1.x, Sqlite3, ejs and [MDL](https://getmdl.io/started)


### Getting started

```
# install
git clone https://github.com/nepaul/web-demos.git
cd simple-koa-app
npm install

# start
npm start
open http://localhost:3000
```
